Game Dev Journey

The beginning

When I was 11 years old, I discovered that a computer could be used for making games and suddenly Unity was installed on my computer ready to make the next GTA, as you can see it never happened, the only thing that happened was that I started to hate coding. I tried several alternatives where you didn't have to code but after that I returned into Unity when I was 13 and resumed my way for becoming the greatest developer of all time (still in progress).

My first finished game

When I was 14 I discovered the term "Game Jam" and felt motivated for participating in the Wowie Game Jam Feb.2019 and there I made my first game in under 72 hours, it doesn't have name because I didn't know you had to put one but there it was, the Wowie Game Jam Feb.2019 Game by Geri which got the 25th place of 52 participants.

Starting my first big and unfinished project

What you have seen is the first and only DevLog of Project Mooba, my first big project and also a mobile game where I inverted a lot of time in developing this huge project, after that video there wasn't much more than 2 weeks of development when I suddenly cancelled the project because of the spaghetti code there was inside of it.
