Simple Tools Project

A Unity package that contains simple tools to use in your project.

While I was in a game jam in 2020, I saw that I was spending too much time programming the exact same tools, so one day I decided to put them all inside a package with the main goal of spending as few time as possible implementing them.

Tile Map Editor

A really basic tile map editor that can export the image and a 2D array of the collision map.

This tile map editor can create a tile map with three different layers and then export all of them or just the ones you want, there you can also create a basic collision map and then export it as a 2D array.
