Researches

👾 Game Development In 16-bits

📹 Yet Another World’s Worst Video Card

2022

A simple research based on Ben Eater’s video series about building a video card on breadboards. In this project I reduce the memory usage by only having 1-bit images.

Made during the Game Jams for the Game Jams, Simple Tools is a bunch of simple and useful tools to use in your Unity projects with the main goal of making it simple and as fast as possible to implement.
