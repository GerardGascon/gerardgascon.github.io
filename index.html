My Games

When I was 11 years old, I discovered that a computer could be used for making games and suddenly Unity was installed in my computer ready to code the next GTA, as you can see it never happened, the only thing that happened was that I started to hate coding. I tried several alternatives where you didn't have to code but after that I returned into Unity when I was 13 and resumed my way for becoming the greatest developer of all time (still in process).

My first game

When I was fourteen I discovered the term "Game Jam" and felt motivated for participating in the Wowie Game Jam Feb.2019 and there I made my first game in under 72 hours, it doesn't have name because I didn't know you had to put one but there it was, the Wowie Game Jam Feb.2019 Game by Geri which got the 25th place of 52 participants.

Starting my first big and unfinished project

What you have seen is the first and only DevLog of Project Mooba my big project and also a mobile game where I put a lot of time in developing this huge project, after that video there wasn't much more than 2 weeks of development when I suddenly cancelled the project because of the spaghetti code there was inside of it.

2020

After the Wowie Game Jam I felt hugely motivated for participating in lots of Game Jams but one year had passed and hadn't developed anything new apart from that weird bullet hell I made for learning.

Isoulated

Something had to be done for having time to spend making games, suddenly a pandemic appeared and then I had as much free time as I ever wanted so no more than one day since I was in my home I saw a Game Jam, that time I wanted to make a game with a team, but didn't find one until the half an hour before the jam started, we were making a game about the social distance but added there some kind of dark creatures and became Isoulated, which won the first place in creativity!!!

Mercorona

Right after the Game Jam had finished, one team needed one programmer, so as an expert I was (spoiler: I wasn't), I decided to join the team, there I learned a whole lot and also I overcame my fear for making 3D games, we tried publishing the game on Google Play Store but for obvious reasons we couldn't. It can now be played here.

Pong-Pong

While we were waiting for Google to reject our game, I joined another Game Jam, this time alone, the LudumDare 46, one of the biggest Game Jams that happen twice a year, as you can see the game is basically Project Mooba but better and developed in less than 3 days instead of the 3 months. It can be played here. By the way, this game got the 614th place over 4959 participants.

The summer

The summer had come and I still wanted to take part in some Game Jams so I made 4 games which were all really basic or were unfinished, the lesson I learned was that you had to make a small game and polish it until the jam ends. The games were FISEC, Train Of Thought, Qu and Spirinaut.
